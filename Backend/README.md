# Smart Credit+ Backend

Smart Credit+ is a secure peer-to-peer (P2P) digital lending platform designed to connect borrowers and lenders within a centralized and regulated environment.

The platform streamlines the entire loan lifecycle - from user registration and KYC verification to loan approval, digital agreements, repayment tracking, and dispute resolution.

By integrating automation, real-time analytics, and administrative oversight, Smart Credit+ enhances transparency, efficiency, and trust in digital financial transactions.

## Technology Stack

Built with [NestJS](https://github.com/nestjs/nest) - A progressive Node.js framework for building efficient and scalable server-side applications.

## Features

- **Admin Module**: Platform administration and oversight
- **Auth Module**: User authentication and authorization
- **Borrower Module**: Borrower management and loan requests
- **Lender Module**: Lender management and loan offerings
- **KYC Module**: Know Your Customer verification
- **Legal Module**: Digital agreements and legal documentation
- **Transactions Module**: Payment and transaction processing
- **Chat Module**: Communication between borrowers and lenders
