# Smart Credit+ Monorepo

This workspace is organized for three main applications:

- `apps/mobile` - React Native app for borrower and lender flows
- `apps/web-admin` - React web frontend for admin operations
- `apps/backend` - NestJS API with Firebase integration

Shared code can live inside:

- `packages/shared-types`
- `packages/shared-constants`
- `packages/shared-ui`

Project docs and supporting files:

- `docs/`
- `scripts/`
